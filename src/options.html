Known Serif Family Names
Used to help classify a page's base typeface as serif. Does not prevent Facade from changing fonts. Default includes PT Serif and mencken-std.
Known Sans Family Names
Used to help classify a page's base typeface as sans-serif. Does not prevent Facade from changing fonts.
Preserved Fonts (Never Replace)
Elements using these fonts will never be restyled. Use for icon fonts (Font Awesome, Material Icons, etc.) that would break if replaced.
FontFace-only Domains
On these domains, the extension loads fonts via FontFace only (no css2 @import or <link>), to avoid strict CSP blocks. Default: x.com.
Inline Apply Domains
On these domains, apply fonts via inline styles to resist CSP and SPA overrides. Default: x.com.
Aggressive Override Domains
On these domains, apply fonts with !important to override strong site styles. Leave empty for normal specificity.
